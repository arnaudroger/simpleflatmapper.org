# [V7.0.0](https://github.com/arnaudroger/SimpleFlatMapper/milestone/201?closed=1)

* [#661](https://github.com/arnaudroger/SimpleFlatMapper/issues/661) Discriminator for 2 association field with same type
* [#655](https://github.com/arnaudroger/SimpleFlatMapper/issues/655) easier to define other enum instantiation type. or any getter `addGetterForType(final Type type, final IndexedGetter<S, T> indexedGetter)`
* [#664](https://github.com/arnaudroger/SimpleFlatMapper/issues/664) allows for non mapped key
* [#659](https://github.com/arnaudroger/SimpleFlatMapper/issues/661) Document how to exclude the jOOQ Open Source Edition as a transitive dependency, when needed
* [#665](https://github.com/arnaudroger/SimpleFlatMapper/issues/661) disable speculative index property finding by default
